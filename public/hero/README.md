# Фото для слайдшоу на главной

Положите сюда 3 файла: `1.webp`, `2.webp`, `3.webp` - они подхватятся слайдером автоматически.

## Рекомендуемые параметры

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Соотношение сторон** | **16:9** | Удобно режется на десктопе и телефоне, блок слайдера тянется по ширине и 50–70% высоты экрана |
| **Разрешение** | **1920×1080** (минимум) или **2560×1440** | Чтобы на Retina не было мыла; 1920 по ширине хватает для большинства экранов |
| **Формат** | **WebP** | Меньший размер при том же качестве; если есть только JPG - положите `1.jpg`, `2.jpg`, `3.jpg` и поменяйте расширения в `components/home/hero-slider.tsx` |

## Важно по кадрированию

Слайдер использует `background-size: cover` и `background-position: center`: картинка масштабируется по меньшей стороне и обрезается по центру.

- На **десктопе** блок широкий → обрежутся верх и низ.
- На **телефоне** блок уже и выше → обрежутся бока.

Поэтому главное (лицо, товар, текст) лучше держать **по центру** и не класть важное в самые края кадра.

## Если используете JPG

Замените в коде слайдера пути с `.webp` на `.jpg` (или добавьте оба варианта и подключайте нужный).
